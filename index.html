  • Boston House Pricing

    Boston House Pricing

    The "Boston House Pricing" dataset is a collection of data that contains information about the housing prices in the city of Boston, Massachusetts. The dataset includes information about the number of rooms in the house, the crime rate in the neighborhood, the distance to schools and major roads, and several other features. The objective of the dataset is to use this information to predict the price of a house in Boston, which can be useful for home buyers and real estate agents. The last column of the dataset contains the price of the house in thousands of dollars.

  • KHR - Keep Healthy and Run

    KHR - Keep Healthy and Run

    KHR is a Java application that aims to efficiently and practically monitor sports teams. KHR is a comprehensive solution designed to help sports teams, coaches, doctors, and players manage and analyze critical data related to player performance, health, and team activities. The KHR application features different login roles tailored to the needs of different users, such as players, coaches, and doctors. This allows them to access specific information and make informed decisions about team sports management.

  • NBA Page

    NBA Page

    Is a web application built in Python using the Streamlit library. This web application performs a web scraping task on a basketball page to gather information about player points in games played from 1950 to 2022. This page use Streamlit's cache to cache the data obtained from web scraping the "basketball-reference.com" website and show it in this page. The web page interface allows users to filter and sort the data according to their preferences. For example, the user can filter the data to show only the points of players in a specific season or in a range of seasons. They can also sort the data according to the number of points scored, the name of the player, or the team they belong to.
